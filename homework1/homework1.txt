+ 1. Скачать контейнер сервлетов TomCat с официального сайта. https://tomcat.apache.org/download-90.cgi

+ 2. Настроить и запустить TomCat согласно инструкции преподавателя на уроке.

3. 
Доработать приложение, которое начал писать на уроке преподаватель, чтобы один из сервлетов выводил список пользователей из репозитория. 
У этот сервлет должен быть доступен по префиксу /users.

4. 
Создать ещё один сервлет, который будет обрабатывать префиксы вида /user/{id}, где {id} - идентификатор пользователя. 
При переходе по ссылке /user/1 должна отображаться информация о пользователе с идентификатором 1.
Для оформления использовать простейшие элементы HTML такие как table и list. Добавлять стили при помощи css не нужно.
